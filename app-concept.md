# The Workout Tracker

The app is called workout tracker.

## Main Screen (Home Screen)
It needs a landing screen.
The useres name is displayed in the upper right corner of the header.
The user's name should be editable.

The landing screen has a button to select a workout.
And a second button to visit previous workouts.
It can show a timer for a currently running workout.
And has a play/pause button if the timer is running.

## Workout Screeen
Every selectable workout contains list of list of excercises.
The workout can be edited (adding removing exercises).

## Excercise Screen
Every exercise can be edited. It contains a name, a descrption and 1..n sets.
Each set can be edited and it contains the weight and the repetition.

## Running Excercise Screen
When a workout is selected it's exercises are displayed: Name, amount of sets, weights, reps.
If clicked it unfolds so every single set can be displayed.
Every set and also the overall exercise can be either checked (completed) or marked as skipped (not done this time).
Every set is editable, so that the amount of sets can be adjusted and the weights and reps in every set can be adjusted.

## General Creation Information
Creation of workouts, exercises, sets should be done with a plus.
Removal with a trash icon.


## Statistics Screen
There should also be statistics screen where the user can select different overviews and graphs.
### Trainings Calender Chart
A list of trainings, when clicked displays details (exercises). A calender like visualizition of trainings in a month.
### Different Trainings Chart
* A list of the workouts sorted by most performed (in selected time frame)
### All Excercises Chart
A bar-chart of excercises. Sortable by
* the amount of times it was done
* the time it was performed
* the weight that was lifted When an excersise is selected a diagram is shown where different graphs can be displayed.
* total weight over time (sum of all reps * weight in all sets)
* max weight
* max wraps
* max total (highest reps * weight result of any set) The dimensions should be time (date when the workout was) on the X achses and the given metric on the Y-Achses
